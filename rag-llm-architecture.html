Enterprise LLM + RAG Architecture

Architecture for enterprise knowledge retrieval using Retrieval Augmented Generation.

User Query ↓ API Gateway ↓ Prompt Orchestration Layer ↓ Embedding Model ↓ Vector Database (Pinecone / FAISS / OpenSearch) ↓ Relevant Context Retrieval ↓ Prompt Augmentation ↓ Large Language Model ↓ Response Generation ↓ Guardrails & Filtering ↓ User Response
